# HR-Employee-Attrition-
## 📌 Overview
This project predicts whether an employee is likely to leave the company using machine learning.

## 🎯 Objective
To help organizations identify at-risk employees and improve retention strategies.

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 🧠 Model Used
Random Forest Classifier

## 📊 Features
- Data preprocessing & encoding
- Feature selection
- Model training & evaluation
- Feature importance visualization

## 📈 Results
- Achieved good accuracy in predicting employee attrition
- Identified key factors influencing employee turnover
